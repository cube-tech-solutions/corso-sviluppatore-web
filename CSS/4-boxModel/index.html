Dimostrazione del Box Model

Box 1

Padding: 20px

Border: 5px

Margin-top e bottom: 30px

Box 2

Margin-bottom: 50px

Box 3

Margin-top: 50px

⚠️ Attenzione: il margine tra Box2 e Box3 sarà solo 50px (non 100px)

✅ Spiegazione:

  • I margini verticali tra Box 2 e Box 3 si "collassano".
  • Anziché sommarsi (50px + 50px), il risultato è il valore maggiore: 50px.
  • Se usi border, padding o overflow: hidden sul contenitore, puoi evitare il collasso.
